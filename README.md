# UX Materno — Projeto Web

## Sobre o projeto
Este projeto foi desenvolvido como atividade acadêmica com o objetivo de aplicar conceitos fundamentais de desenvolvimento web utilizando HTML e CSS.

A proposta aborda a relação entre tecnologia, maternidade e experiência do usuário (UX), com foco na criação de interfaces mais humanas, acessíveis e intuitivas.

---

## Tecnologias utilizadas
- HTML5  
- CSS3  

---

## Páginas do projeto

- index.html → Página inicial com textos, listas e navegação  
- pagina1.html → Galeria de imagens  
- pagina2.html → Tabelas, mídias e formulário  

---

## Objetivos

- Aplicar conceitos básicos de HTML  
- Utilizar CSS para estilização  
- Criar navegação entre páginas
